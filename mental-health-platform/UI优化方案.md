# 心理健康平台 UI 优化方案

## 设计理念
参考韩国现代UI设计风格，打造温暖、专业、易用的心理健康平台

## 核心设计元素

### 1. 色彩系统
- **主色调**：温柔粉色 #FF6B9D（温暖、友好、舒适）
- **辅助色**：
  - 清新薄荷绿：#7DD3C0
  - 温暖珊瑚橙：#FFA07A
  - 天空蓝色：#A8D8EA
- **中性色**：
  - 深色文字：#2D3748
  - 次要文字：#718096
  - 背景色：#F7FAFC
  - 卡片背景：#FFFFFF
  - 柔和背景：#FFF5F7

### 2. 圆角和阴影
- 小圆角：8px
- 中圆角：12px
- 大圆角：20px
- 卡片阴影：0 4px 20px rgba(0, 0, 0, 0.08)
- 悬浮阴影：0 8px 30px rgba(0, 0, 0, 0.12)

### 3. 间距系统
- 基础间距：8px
- 小间距：12px
- 中间距：16px
- 大间距：24px
- 超大间距：32px

### 4. 字体系统
- 标题：24px-32px，字重 700
- 副标题：18px-20px，字重 600
- 正文：14px-16px，字重 400
- 小字：12px，字重 400
